Ink Controls
Give a vibe and your proper dither will populate
Every control stays editable for every preset so you can build your own mixes: tune grid, bridges, contrast, gamma, etc., then switch algorithms — values are kept in the panel (only the Threshold row hides when it does not apply). The preview always uses the controls that matter for the current algorithm. Error diffusion uses every slider: Grid (1–140, 140 = full width), radii (blur), bridges/tone/gradient (edge + error shaping), stipple jitter / edge strength (noise & edge weight), plus contrast / quantize / gamma / taper. Remixed in the status line means your numbers differ from the preset file in presets.js.
